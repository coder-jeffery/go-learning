http和https区别：
    http 默认 80端口
    https 默认 443端口  ssl/tls证书 CA机构办法 https://  加密解密 
        ssl/tls证书：
            协议版本+加密套件+随机数+服务器证书
            客户端->服务端：协议版本+加密套件+随机数A
            服务端->客户端：协议版本+加密套件+随机数B+服务器证书
            密钥交换： 随机数A+随机数B+预主密钥 +对称加密

常用加密算法：
    对称加密：
        AES，DES，3DES，RC4
    非堆成加密：
        RSA，ECC，DSA
    哈希算法：
        MD5，SHA-1，SHA-256，SHA512
