golang基础知识：
    golang闭包特点：
        1 函数作为返回值/参数传递
        2 内部函数访问外部函数的局部变量
        3 外部变量生命周期会被延长，与闭包绑定
    常用的状态码有哪些？
        10x 
        20x 返回成功
            200 查询成功
            201 创建成功
        30x 重定向问题
            302 临时重定向
            304 缓存未修改 使用本地缓存
        40x 客户端问题
            401未登录 / 403无权限 / 404未找到页面
            400 bad request 请求语法错误
            401 unauthorized 未认证
            403 forbidden 权限不足 拒绝访问
            405 请求访问不对 get/post
        50x 服务端问题
            500服务器宕机 internal server error 服务器内部错误
            502 bad getway 网关报错
            503 service unavaliable 服务不可用
            504 getway超时 网关超时
幂等性：

    get/put/delete 幂等接口
    post 非幂等性：
